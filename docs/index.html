How to install in Chrome

Как установить в Chrome

1
Download and unzip Скачай и распакуй Click the button above to download the ZIP file, then extract it to any folder on your computer. Нажми кнопку выше, скачай ZIP-архив и распакуй его в любую папку на компьютере.
2
Open Extensions page Открой страницу расширений In Chrome, navigate to chrome://extensions or go to Menu → More tools → Extensions. В Chrome открой chrome://extensions или перейди в Меню → Дополнительные инструменты → Расширения.
3
Enable Developer mode Включи режим разработчика Toggle the Developer mode switch in the top-right corner of the Extensions page. Переключи тумблер Режим разработчика в правом верхнем углу страницы расширений.
4
Load unpacked Загрузи распакованное расширение Click Load unpacked and select the folder you extracted in step 1. Done! Нажми Загрузить распакованное и выбери папку из шага 1. Готово!
5
Pin the extension Закрепи расширение Click the puzzle icon in Chrome toolbar, find MarkDownload and click the pin icon to keep it visible. Нажми иконку пазла на панели Chrome, найди MarkDownload и нажми иконку закрепления.

Works on Chrome 90+, Edge, Brave, and any Chromium-based browser. Also available for Firefox via Mozilla Add-ons.
Работает в Chrome 90+, Edge, Brave и любом браузере на Chromium. Также доступно для Firefox через Mozilla Add-ons.
